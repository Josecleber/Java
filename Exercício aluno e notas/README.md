# Exercício: Aluno e Notas (Java)

Projeto de aluno com:

- Nome
- Matrícula
- Vetor de notas
- Cálculo da média
- Impressão de informações

---

## Estrutura

| Arquivo | Descrição |

| Aluno.java | Representa o aluno e suas operações |

| AppAluno.java | Programa principal para criar aluno e mostrar suas informações |
