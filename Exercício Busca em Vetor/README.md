# Exercício: Busca em Vetor (Java)

Este programa:

- Gera um vetor com 15 números aleatórios (0 a 99)
- Exibe o vetor gerado
- Pede ao usuário um número para buscar no vetor
- Mostra se o número foi encontrado e em qual posição
