# Exercício: Carro (Java)

Projeto que:

- Cria uma classe `Carro`
- Possui atributos:
  - Marca (String)
  - Ano (int)
- Método para exibir informações do carro

---

## Estrutura

| Arquivo | Descrição |

| Carro.java | Representa o carro |

| AppCarro.java | Cria e exibe o carro |
